# Mock Exam 2 - Question 12

## Question 12: Deployment, Service, and Network Policy

Please use the namespace `nginx-deployment` for the following scenario.

**Tasks:**

1. Create a deployment with name `nginx-ckad11` using `nginx` image with **2 replicas**
2. Expose the deployment via **ClusterIP** service named `nginx-ckad11-service` on port **80**
3. Use the label `app=nginx-ckad` for both resources
4. Create a NetworkPolicy named `ckad-allow` so that only pods with label `criteria: allow` can access the deployment on port **80**

## Solution

### Step 1: Create Deployment and Service

```bash
kubectl apply -f - <<EOF
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-ckad11
  namespace: nginx-deployment
spec:
  replicas: 2
  selector:
    matchLabels:
      app: nginx-ckad
  template:
    metadata:
      labels:
        app: nginx-ckad
    spec:
      containers:
        - name: nginx
          image: nginx
          ports:
            - containerPort: 80
---
apiVersion: v1
kind: Service
metadata:
  name: nginx-ckad11-service
  namespace: nginx-deployment
spec:
  selector:
    app: nginx-ckad
  ports:
    - name: http
      port: 80
      targetPort: 80
  type: ClusterIP
EOF
```

### Step 2: Create Network Policy

```bash
kubectl apply -f - <<EOF
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: ckad-allow
  namespace: nginx-deployment
spec:
  podSelector:
    matchLabels:
      app: nginx-ckad
  ingress:
    - from:
        - podSelector:
            matchLabels:
              criteria: allow
      ports:
        - protocol: TCP
          port: 80
EOF
```
